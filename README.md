# DTS React User Guide

## Install

```sh
npm install react-web3-icons  # for npm users
yarn add react-web3-icons     # for yarn users
pnpm add react-web3-icons     # for pnpm users
```

## Quick Start

```ts
import { Bitcoin } as icn from '../.';

function App() {
  return (
    <div>
      <Bitcoin />
    </div>
  );
}
```
